Django Gears
============

This application provides integration with `Django`_ and `Gears`_. Gears is
a Python application that compiles and concatenates JavaScript and CSS
assets. Inspired by Ruby's Sprockets.

.. _Django: http://www.djangoproject.com
.. _Gears: http://gears.readthedocs.org/en/latest/

Source code is available on github: https://github.com/gears/django-gears
